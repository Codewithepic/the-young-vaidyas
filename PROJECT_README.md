# The Young Vaidyas - Ayurvedic Wellness Website

A modern, professional Ayurvedic wellness website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Responsive Design**: Mobile-first approach with clean, modern UI
- **Ayurvedic Consultation System**: Booking interface for wellness consultations
- **Product Showcase**: Display of authentic Ayurvedic formulations
- **Founder Profiles**: Information about the BAMS scholars behind the brand
- **Content Management**: Blog system and customer testimonials
- **SEO Optimized**: Built with performance and search engines in mind

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS with custom design system
- **Fonts**: Inter (sans-serif) and Playfair Display (serif)
- **Icons**: Custom SVG icons and emoji fallbacks

## Color Palette

- **Primary**: Earthy greens (#3d8b5c to #f0f9f4)
- **Secondary**: Warm browns (#c8732b to #fdf8f3)
- **Accent**: Golden yellows (#eab308 to #fefce8)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── app/
│   ├── globals.css          # Global styles and Tailwind imports
│   ├── layout.tsx           # Root layout component
│   └── page.tsx             # Home page
├── components/
│   ├── Header.tsx           # Navigation header
│   ├── Hero.tsx             # Hero section
│   ├── FeaturedProducts.tsx # Product showcase
│   └── Footer.tsx           # Site footer
```

## Content Integration

The website integrates content from:
- `products.json` - Product catalog
- `founders.json` - Founder information
- `site-info.json` - Site metadata and navigation
